DESCRIPTION

r.viewshed.fuzzy creates fuzzy viewsheds where the level of visibility decays with distance (see references for more information). The level of decay is determined by the visual arc and the size of the target looked at. The threshold for the 'optimum' level visibility is set with the 'dist_b1' parameter. The denominator parameter controls how steep the visibility drop-off is. The output raster returns cells with values ranging from 1-0. The closer the value is to 1/0, the higher the level of clarity.

NOTES

The module uses r.viewshed. It requires the same inputs, and can use the same flags. It also uses r.grow.distance create distance decay raster values. After finishing the operations, the module does not load automatically the output(s) for display. For the moment only works manually.

EXAMPLE

With default parameters and flags -c (consider curvature of the earth) and -r (consider the effect of atmospheric refraction) enabled, target width set to 1 m, visual arc to 1 minute, and distance to b1 1 km:
r.viewshed.fuzzy -c -r vect=singlePt rast=DEM output=singleTest twidth=1.0 varc=1.0 dist_b1=1000
With default parameters and flags -c (consider curvature of the earth) and -r (consider the effect of atmospheric refraction) enabled, target width set to 3 m, visual arc to 0.5 minute, and distance to b1 1 km:
r.viewshed.fuzzy -c -r vect=multiPt rast=DEM output=multiTest twidth=3.0 varc=0.5 dist_b1=1000
